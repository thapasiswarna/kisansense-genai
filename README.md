KisanSense GenAI is an AI-powered agricultural advisory system designed to help farmers with crop diseases, pest control, fertilizer usage, and government scheme queries.

## Features
- Natural language farming queries
- Offline semantic search (FAISS)
- AI-enhanced advisory using Generative AI
- Simple and accessible Streamlit interface

## Tech Stack
- Streamlit
- Python
- FAISS
- Sentence Transformers
- Google Gemini API

## Deployment
Deployed on Streamlit Community Cloud
